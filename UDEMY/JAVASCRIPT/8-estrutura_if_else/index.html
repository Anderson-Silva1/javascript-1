IF e Else

É uma estrutura de descisão que o JavaScript posui

If significa "se" e else "se não"... Trocando em miudos, será execultado o bloco de código if caso a consição do if seja verdade e mesma coisa acontece com o else...

Temos também o Else If, que significa "se, se não", e vai ser usado como uma segunda condição do if
