JavaScript no HTML

Temos duas formas de colovar JS no HTML, forma interna e externa


INTERNA:

Usamos uma tag com fechamento dentro do HTML chamada script e dentro dela o nosso código JS

        < script >
            "código JS"
        < /script >

EXTERNA:

Usamos uma tag com fechamento dentro do HTML chamada script com o atributo "SRC" levando até o arquivo JS onde contém nosso código JS

        < script src="./script.js" > < /script >


Qual usar? Interno ou Externo?

Depende do contexto, mas o aconselhável é sempre criar um arquivo externo para desenvolvimento, pois assim temos um maior controle do nosso projeto, sem contar que podemos ter mais de um tipo de arquivo ".js" dentro do nosso HTML



Devemos ter cuidado com a precedência dos nossos Arquivos JS

Precedência é basicamente qual a ordem de execução que eu quero na minha ágina HTLM...

Os Browsers conseguem ler os códigos enviados a ele de cima para baixo, ou seja, o que estiver nas primeiras linhas será execultado primeiro e vice-versa

Se colocarmos o JS para carregar antes dos nossos elementos própriamente dito, e se caso o nosso JS for selecionar algum elemento, ele nãi vai existir, ainda, pois ainda não foi execultado pelo Navegador...
