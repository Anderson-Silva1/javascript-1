Operadores

Operadores decomparação

  • == (IGUAL) - Verifica se os valores são iguais
  • === (IDÊNTICOS) - Verifica se os valores são idênticos, tanto em caractere quanto em tipo
  • != (DIFERÊNTE) - Verifica se os valores são Diferentes
  • !== (NÃO IDÊNTICO) - Verifica se os valores são Não Idênticos
  • > (MAIOR QUE) - Verifica se o primeiro valort é maior que o segundo
  • >= (MAIOR OU IGUAL A) - Verifica se o primeiro valor é maior ou Igual ao segundo
  • < (MENOS QUE) - Verifica se o primeiro valor é menor que o segundo
  • <= (MENOR OU IGUAL A) - Verifica se o primeiro valor é menor ou Igual ao segundo


Operadores Lógicos

  • && (E) - Verifica mais de uma condição, se uma delas for false, o resultado sará false, se todas as consções forem True, retornará True
  • || (Ou) - Verifica mais de uma condição, se uma for True e a outra False o resultado será True
  • ! (Negação) - Verifica qual o resultado da condição e inverte, caso dê True o resultado será false, caso dê False o resultado será True

Operadores Ternários

Seria uma forma simplificada de usar o IF e o ELSE

Nele usamos Os caracteres: "?" e ":"

  • Sintaxe

    < condição > ? < vedadeiro > : < falso >

    Basicamente esse código está dizendo o seguinte: SE "consição" for "verdade", vai execultar o bloco de código após o "?", SE "condição" for "falsa", vai execultar o bloco de código depois do ":"


    EX: 5 > 2 ? console.log("Sim") : console.log("Não")

  • Os Operadores Ternários devem ser ultilizados para coisas simples, pois dentro dele não conseguimos fazer coisas mais complexas
