Comentário

Basicamente, esta é uma funcionalidade que todas as linguagens de programação tem, onde tem a funcionalidade de criar linhas de códigos que não serão execultadas...

Os comantários são usados para debugar nossa aplicação, ou para explicar o que cada trecho de código faz

Dentro do JS temos 2 tipos de comentários:

  • Comentário de 1 linha

    No JS será definido com o "//" (Olhar exemplo no código fonte)

  • Comentário de multiplas linhas

    No JS será definido com o "/* */" (Olhar exemplo no código fonte)
