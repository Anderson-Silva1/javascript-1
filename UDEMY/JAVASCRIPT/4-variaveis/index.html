Variáveis

Variáveis são espaços reservados na memória do computador para armazenar um determinado tipo de dado

Tipos de variáveis

Temos pelo menos 3 tipos de variáveis no JS:

  • VAR

    Que é um tipo de variável Global e volátil

  • LET

    Que pe um tipo de variável Local e volátil

  • CONST

    Que pe um tipo de variável Local e não NÃO volátil



  • GLOBAL = É um tipo de variável que pode ser acessado de qualquer lugar da nossa aplicação
  • LOCAL = É um tipo de variável que só pode ser acessada dentro de um escopo específico
  • VOLÁTIL = Significa que ela pode ser alterada ao longo da nossa aplicação
  • NÃO VOLATIL = Significa que ela não pode ser alterada ao longo da nossa aplicação


Valores de variáveis

Podemos ter alguns tipos de valores para variáveis, e aqui estamos falando já de tipos de dados, vamos mostrar os 3 principais tipos de dados

  • STRING

    São dados do tipo "textos"

    Estes são especificados pelas áspas simples ou áspas duplas... Veremos mais afundo sobre eles mais para frente...

  • NUMBER

    São dados do tipo "numero"

    Podem ser especificados como "INT" e "FLOAT"

    • INT = Números inteiros: 1, 2, 3, 4, 5, etc...
    • FLOAT = Números decimais ou fracionados: 1.0, 1.1, 1.2, 1.3, 1.4, etc...

  • BOOLEAN

    São dados que darão apenas 2 resultados possíveis:

    • TRUE = Verdadeiro
    • FALSE = Falso

  • NULL

    é um valor especial que representa a ausência intencional de um valor para uma variável ou objeto. Ele é usado para indicar que uma variável foi declarada, mas ainda não foi atribuída a nenhum objeto ou valor.

  • UNDEFINED

    é um valor primitivo que indica que uma variável foi declarada, mas ainda não foi inicializada com um valor

    Também pode ser retornado em outros contextos, como quando você tenta acessar uma propriedade de um objeto que não existe ou quando uma função não retorna explicitamente um valor.



Regras de declaração de variável

1 - NÃO pode ser inicuada por números, apenas letras ou "_"

2 - NÃO pode conter caractéres especiais como: ç, ~, ^, etc...

3 - NÃO pode ser igual a palavras reservadas da linguagems

  • PALAVRAS RESERVADAS = São palavras que a Linguagem de programação usa para sua sintaxe, alguns exemplos que podemos usar em relação ao JS são: if, else, else is, for, while, switch, var, let, const, case, break, catch, await, finaly, true, false, null, new, e mais uma infinidade de outras palavras...



Informações importantes

O JavaScript possui a tecnologia CASE-SENSITIVE

Case Sensitive = Significa que o JavaScript faz distinção a formas de escrita, ou seja, para o JS a palavra "teste", "Teste" e "TESTE" são todas diferentes, e podem receber diferentes valores dentro delas, "teste" pode receber um "boolean", "Teste" póde receber um "number int" e "TESTE" pode receber uma "string"


Existe uma convenção de escrita para declaração de variável no JavaScript:

  • camelCase = seria escrever nossas variáveis na forma de camelo, ex:

    • Vamos declarar uma variável para armazenar o nome de uma pessoa, ficará mais ou menos assim: personName

  • snake_case = Seria escrever nossa variável na forma de uma cobra, ex:

    • Vamos declarar uma variável para armazenar o nome de uma pessoa, ficará mais ou menos assim: person_name

Obs: Sempre ao declararmos nomes para as variáveis, devemos declará-las em inglês
