Introdução JavaScript

JavaScript é uma linguagem de programação de alto nível ( Está mais próximo a linguagem Humana )

Podemos usar o JavaScript para criar aplicações do tipo:

  • "Client-side" (Lado Cliente, mais conhecido como Front-End)
    • OBS:Usando o JavaScript em Client-side, ele vai trabalhar mais para tornar a página interativa
  • "Server-side" (Lado Servidor, mais conhecido como Back-End)
  • "Desktop" (Aplicativo projetado para ser instalado e execultado dentro de um computador)
  • "Mobile" (Aplicativo projetado para dispositivos móveis)

JavaScript NÃO é JAVA

JavaScript é um tipo de Linguagem de Programação Script, o que significa que ele não precisa de um compilador ou fazer o processo de Build, o próprio browser consegue ler e execultar códigos JavaScript's
