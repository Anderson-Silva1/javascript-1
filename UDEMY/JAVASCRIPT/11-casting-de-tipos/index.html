Casting de tipos

Casting seria uma conversão de tipos de dados... Podemos usar 3 tipos de casting's: : toString(); ParseInt(), ParseFloat()

  • toString() ≫ Converte para Texto
  • ParseInt() ≫ Converte para Número Inteiro
  • ParseFloat() ≫ Converte para Número Decimal ou Fracionado
