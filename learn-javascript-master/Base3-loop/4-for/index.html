For

Diferente do WHILE, que recebe apenas uma expressão para verificação, o for recebe mais duas coisas:

  • VARIÁVEL
  • EXPRESSÃO
  • AÇÃO DE CONTROLE

Sintaxe:

        for( let count = 0 /* VARIÁVEL */; count < 10 /* EXPRESSÃO */; count++ /* AÇÃO DE CONTROLE */){
            console.log(count) // BLOCO DE CÓDIGO
        }
