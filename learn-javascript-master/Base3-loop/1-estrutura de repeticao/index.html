Estrutura de repetição

São expressões que são ultilizadas para poder repetir um determinado bloco de códigos... Pode ser determinado, ou seja, com um limite estabelecido ou até uma condição ser atendida... O mais conhecidos são: While, For e Do-While

Loop é uma interação com um determinado conjunto de código, sendo de forma definida, ou indefinida (até uma espressão atender o valor necessário)
