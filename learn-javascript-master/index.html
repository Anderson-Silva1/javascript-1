JavaScript

É uma linguagem de programaçõa de alto nível multi-paradigma de tipagem dinâmica e fraca... Ele surgiu no ano de 1995 pelo Brendan Eich a serviço da NETSCAPE com o intuito de tornar as páginas web mais dinâmicas e interagíveis com o usuário

Paradigma: É a forma como a Liguagem fornece os recursos para a gente desenvolver uma solução... Realacionado ao JavaSript podemos dizer que ele tem o paradigma ORIENTADO A OBJETOS, FUNCIONAL, ORIENTADA A EVENTOS...

Inicialmente ele foi criado para a WEB, mas hoje, com a ajuda do NODE JS, o JavaScript pode ser usado do lado servidor também...
