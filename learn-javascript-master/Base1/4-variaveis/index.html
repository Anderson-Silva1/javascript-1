Variáveis

São conteiners de armazenamento de dados... Temos 3 tipos de variáveis em JS: VAR, LET e CONST

VAR = É uma variável do tipo global e volátil, ou seja em qualquer parte do nosso código era possível mudar o tipo de dado dessa variável... Era a forma tradicional de se declarar uma variável no JS antes da versão EC8, esse tipo de declaração não deve mais ser usado

LET = É um tipo de variável por escopo... o que significa que ela pode ser usada apenas dentro do escopo de criação dele... É uma tipo de variável volátil... Porém só pode ser modificada dentro do seu escopo de origem

CONST = É um tipo de variáavel constante, ou seja, não pode ser modificada, só pode receber um valor uma vez...
