Comentário

Comentários são trechos de códigos que serão ignrados pelo Browser (navegador)...

Em JavaScript eles são representados de duas formas: De uma (1) linha ou de 2 (duas) ou mais linhas

  • 1 Linha => É reprensentado pelo "//"
  • 2 ou mais linhas é representado pelo "/**/"

Exemplo

        function sum(a,b){ // Esse é um comentário de uma linha e não vai aparecer no navegador... Se eu pular de linha como ENTER
            vai aparecer essa parte do texto, e irá dar um erro
            const soma = a + b
            return soma
            
            /*

                Tanto aqui

            */
