Importando JavaScript no HTML

Podemos importar JAs de duas formas: JS interno e JS externo

JS interno:

No JS interno significa que iremos importar nosso arquivo JS dentro da nossa página HTML, tanto dentro do HEAD, quando dentro do BODY quando depois do BODY, ficando com a sintaxe mais ou menos assim:

        <html>
            <head>
            <script>
                
                    // Podemos colocar ele aqui
                    /* Aqui dentro ficará nosso código JavaScript */
                
            </script>
            </head>
            <body>
            <h1> Meu projeto </h1>
            <script>
                
                    // Podemos colocar ele aqui
                    /* Aqui dentro ficará nosso código JavaScript */
                
            </script>
            </body>
            <script>
                
                    // Podemos colocar ele aqui
                    /* Aqui dentro ficará nosso código JavaScript */
                
            </script>
        </html>

JS externo

Essa é a forma mais recomendada de usar um arquívo JS... Nessa modalidade usamos a sintaxe: <script src="script.js"> </script>... e podemos cvolocá-la dentro do HEAD, dentro do BODY ou depois do BODY do nosso arquivo HTML... mas para evitar futuros erros de carregamento, usamos geralmente no final do BODY ou depois do BODY

¨ 
        <html>
            <head>
            <script scr="script.js"></script>
            </head>
            <body>
            <h1> Meu projeto </h1>
            <script scr="script.js"></script>
            </body>
            <script scr="script.js"></script>
        </html>
