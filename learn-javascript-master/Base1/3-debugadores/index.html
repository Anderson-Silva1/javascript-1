Debugadores

São ferramentas que usamos para testar o nosso sistema

Temos o "console.log()" que vai nos retornar uma mensagem, sintaxe: console.log("Teste")

Temos o "console.info()" que vai nos retornar uma mensagem diferente da de cima, sintaxe: console.info("Teste")

Temos o "console.warn()" que vai nos retornar uma mensagem de alerta, sintaxe: console.warn("Teste")

Temos o "console.error()" que vai nos retornar uma mensagem erro, sintaxe: console.error("Teste")


Usamos também o "alert()" que vai nos retornar uma caixa de texto, sintaxe: alert("Olá")


Usamos também o "confirm()" que vai nos retornar uma caixa de texto, sintaxe: confirm("Aguardando confirmação")


Usamos também o "prompt()" que vai nos retornar uma caixa de texto, sintaxe: prompt("Qual o seu nome?")
