Introdução POO (Programação Orientada a Objetos)

Assim como os Arrays, o POO é uma estrutura de dado, e esses dados, diferente dos dados do Array que é organizado em lista, é organizado por propriedades, Associando chave e valor, onde chave é o nome da propriedade e valor é o valor que essa propriedade irá receber... Mas assim como os Arrays, o POO pode armazenaroutros tipos de dados dentro dele
