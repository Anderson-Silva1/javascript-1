Condicionais

São blocos de códigos que são execultados baseado no resultado de uma expressão lógica. Também são chamada de Controle de Fluxo

IF

If significa "se" e pode ser usado para definir um fluxo... Ele vai verificar um bloco de código e se o resultado for true ou verdadeiro ele vai execultar outro Bloco de código

Junto com o IF, podemos usar o "else" que significa "se não"... ele vai ser execultado quando o IF não satisfazer o resultado desejado... Vale resaltar que o IF funciona sozinho sem o ELSE mas o ELSE não funciona sem o IF

Temos tbm o ELSE IF que é basicamente o IF dentro do ELSE, que significa: "Se não, se"
