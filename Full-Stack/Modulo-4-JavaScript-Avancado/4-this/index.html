Aprendendo sobre o this

Antes de entender sobre o this precisamos saber o que é escopo

Escopo é um local dentro do código... No código temos 2 escopos: Global e um Local

Global é tudo aquilo que fica na fonte do nosso projeto

Local é tudo que está dentro de algum outro localdentro do escopo Global... Como exemplo uma variável let ou const dentro de um IF, está dentro de um escopo Local

É interessante saber que dentro do escopo Global possui um objeto chamado global, que contém diversas ferramentas, e uma delas é o console.log... Onde o console é uma propriedade do objeto Global e um objeto ao mesmo tempo, e o LOG é uma propriedade do objeto console
