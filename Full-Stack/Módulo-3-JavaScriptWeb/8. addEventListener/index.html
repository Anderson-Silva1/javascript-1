Vamos aprender sobre o Event listening

Essa é uma função em JavaScript que ouve a função de algum evento e faz algo com ele, ou seja, ela vai ficar esperando o evento desejado acontecer para ela avisar ou rodar algum outro código

Nesse modo de utilizar os eventos fazemos assim:


button.addEventListener('click', function() { console.log("Este botão foi clicado"); })


Onde pegamos uma variável que já tem um elemento HTML armazenado em si, junto com o ".addEventListener()" ... E dentro desses parênteses estarão duas instruções separadas por ",", onde a primeira instrução é o evento que vai acontecer (sem o "on" na frente, por exemplo: click) e a segunda instrução é o que o "addEventListener" vai fazer quando esse evento acontecer.

Nesse caso, quando o botão tiver o evento de "click", ele vai dar um console.log("Este botão foi clicado")
