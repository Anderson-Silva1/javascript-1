Buscando atributos de um elemento

podemos pegar os atributos de um elementos HTML usando o ". + o atributo" em JS de

Exemplo:

        
const teste = document.querySelector('input')

console.log(teste.type)

Nesse caso, declaramos uma variável em javascript e chamamos ela para uma função chamada console.log(), que serve como uma especie de debugador ou um testador dos nossos códigos, puxando o atributo "type"
