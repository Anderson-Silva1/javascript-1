Vamos aprender sobre as ptropriedades de eventos

Podemos ver as Propriedades de um evento usando o seguinte codigo em JS com bordas azuis:

        
const botao_teste = document.querySelector('#button-test')

botao_teste.onclick = (propriedades) => {
    console.log(propriedades);
}

Esse código está colocando dentro de uma variável o nosso ID "button-test" e atribuindo a ele uma função onde os parâmetros dela são as pripriedades do próprio botão... "propriedades" são os parâmetros da nossa função e eles são chamados no "console.log(propriedades);", e essas pripriedades nós conseguinmos ver de forma individual para os mais diversíficados ramos existentes no mundo usando apenas o "." de pois do no0sso parâmetro ficando assim: console.log(propriedades.key)
