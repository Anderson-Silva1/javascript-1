ESTRUTURA DE DECISÃO:
    IF:
        - São estruturas que tem a capacidade de descidir(mediante a uma condição) o caminho que o código irá percorrer...
        - Temos a principal estruturade decisão em JavaScript o IF( < CONDIÇÃO > ) { < BLOCO DE CÓDIGO > }
        - IF significa "se"
        - Temos o ELSE que significa "se não"
        - E temos também o ELSE IF que significa "se não, se"
        - Sintaxe veja no código

    SWITCH:
        - Ele também é uma estrutura condicional, ou seja ele também serve para controlar o fluxo... Ele vai avaliar uma expressão e procurar a melhor cláusula (caso) que atenda ao resultado
        - Dentro dele os casos ou cláusulas são chamados de cases, onde cada case é acompanhada do break que é uma forma de parar o código caso chegue a um resultado aceitável
