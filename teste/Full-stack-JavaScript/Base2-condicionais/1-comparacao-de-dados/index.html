Comparação de dados

São operações que contém o resultado boleano, verdadeiro ou falso, true ou false...

sinal = "=="

Significa igualdade, EX:

        const velocity = 90
        console.log(velocity == 90)

resultado: True


Ainda pelo sinal "=="

Como já foi explicado o JS tem um atipagem dinâmica, ou seja, ele consegue mudar o tipo de dado com mais facilidade

        const velocity = 90
        console.log(velocity == "90")

Nesse caso estamos comparando o number valor 80 com a string valor 80, que dará o resultado True, ou seja com esse sinal ele não considera o tipo de dado...


Sinal = "==="

Tambvém significa igualdade, porém é uma igualdade mais específica

        const velocity = 90
        console.log(velocity === "90")

Esse resultado dará false pois o tippo de dado é diferente


Sinal = "!="

Este sinal significa DIFERENTE

        const velocity = 90
        console.log(velocity != 80)

Como resultado dará true, pois 90 é diferente de 80...


Ainda sobre o sinal = "!="

        const velocity = 90
        console.log(velocity != "90")

Perceba que neste exemplo dará o valor false, onde está dizendo que 90 não é diferente de "90"

Podemos usar o diferente específico, que difere também o tipo de dado

        const velocity = 90
        console.log(velocity !== "90")

Dessa forma ele irá distinguir o tipo de dado e conseguirá diferenciar, ou seja, dará o valor true pois 90 é diferente de "90"


Sinal = "<"

Este sinal simboliza "Menor que"

        const velocity = 90
        console.log(velocity < 100)

Teremos a resposta true pois 90 é menor que 100


Sinal = ">"

Este sinal simboliza "Maior que"

        const velocity = 90
        console.log(velocity > 100)

Teremos a resposta False pois 90 não é maior que 100

Sinal = "<=" ou ">="

Estes sinais simbolizam: Menor ou igual a... ou Maior ou igual a.... Respectivamente...
