Funções anônimas

Podemos Criar alguns tipo de Funções diferente do padrão que é: function teste(){return}

Podemos criar funções anonimas... Que são funcões sem nome...: function(){return}, mas dessa forma dará um erro

O que popdemos fazer é armazenar essa função dentro de uma variável, para atribuir o valor dessa função dentro da variável: let teste = function(){return}
