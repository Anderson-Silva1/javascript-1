Parâmtros Padrôes

Os parâmetros padrôes servem para quando um parâmetro não for especificado na chamada de uma função, não dar um erro ou um valor undefined

        function sum (a = "Número 1 não especificado",b = "Número 2 não especificado"){
            const sum = a + b
            console.log(sum)
        }
        
        sum(1)
