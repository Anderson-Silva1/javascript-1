Operações básicas

Podemos usar em relação aos tipos numbers a soma (+), subtração (-), divisão (/), multiplicação (*), resto ou módulo (%)

Podemos usar em relação aos tipos Strings a concatenação, que é juntar dois dados String usando o sinal de adição (+): "Anderson" + "Silva" = AndersonSilva

Podemos ainda em relação aos tipos Strings pegar caracteres específicos usando o índex ou posição que é especificado pelos colchetes []: "Anderson"[2] = d... Estamos pegando o índice 2 do dado tipo String "Anderson" e estamos recebendo a letra "d"... Pois na programação geralmente não iniciamos contando do 1 (um) e sim do 0 (zero)... Ou seja o índice ou posiçõa 2 será a Terceira letra da nossa string, o que aconcece o mesmo com uma lista, mas isso é questão para outro momento...

Podemos ainda em relação aos tipos Strings saber o tamanho dela usando a operação length: "Anderson".length = 8... temos 8 caracteres no nome Anderson... essaq operação pega o tamanho do nosso dado, no caso string e contou quantos caracteres ela tinha, no caso de uma lista vai funcionar da mesma forma...
