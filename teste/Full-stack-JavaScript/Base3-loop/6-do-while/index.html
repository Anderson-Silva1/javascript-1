Do While

"Do While" é parecido com o "While", a única diferênça é que o "Do While" vai execultar o código pelo menos uma vêz antes da expressão

Sintaxe:

        let velocity = 110
        let acceleration = 5

        do {
            console.log(`Acelerando: estamos a ${velocity} km/s`)
            velocity += acceleration
        } while(velocity <= 100)

Nesse código "velocity" não é menor e nem igual a 100, "velocity" é 110... Mas perceba que o código será execultado, e o "velocity" será 110, o que está fora da expressão de validação "enquanto"(while)... Ou seja, o "Do While" vai execultar o código pelo menos uma vêz independente da expressão
